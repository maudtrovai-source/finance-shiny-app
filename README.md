# Real Estate Financial Dashboard

A web application built with Python and Shiny designed to process, track, and visualize financial data. 

## Project Overview
This project features a secure SQLite backend and an interactive dashboard to simulate cost and valuation data for specific development profiles, such as calculating material and labor projections for a 991 square meter residential stand.

## Technologies Used
* **Language:** Python
* **Framework:** Shiny
* **Database:** SQLite
* **Visualization:** Plotly & Pandas
